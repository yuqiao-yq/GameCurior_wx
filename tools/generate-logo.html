💡 使用说明

1. 上方切换主题色，3 个方案会实时更新。

2. 选好后点击"下载 PNG（512×512）"，文件保存到本地。

3. 推荐使用场景：

　　- 小程序图标：在微信公众平台后台 → 基本设置 → 头像上传（要求方形 ≥ 144×144，512 足够）

　　- 小程序启动图：可直接用作 Hero 区域 logo

　　- App.json 中的 tabBar：本工具已有 tools/generate-tabbar-icons.html 单独生成

4. 如需 1024×1024 或矢量 SVG，可联系开发者扩展工具脚本。
